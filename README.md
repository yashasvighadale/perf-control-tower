# Perf Control Tower

A standalone HTML dashboard for analyzing Meta Ads performance across Sep 2025 – Feb 2026.

## What's Inside

`Meta_Ads_Command_Center.html` — an all-in-one, single-file analytics dashboard that includes:

- KPI summary cards (spend, ROAS, CTR, CPA, etc.)
- Audience and creative performance breakdowns
- Filterable tables and tabbed views
- Automated insights and verdict labels (best/worst performers)

## Usage

No dependencies or server required. Just open the file in any browser:

```bash
open Meta_Ads_Command_Center.html
```

## Tech

Pure HTML/CSS/JS — no frameworks, no build step.
